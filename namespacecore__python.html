Detailed Description

A collection of Python interface helpers.

These functions provide an interface between the types in core_types and native Python types.

They should be used in the interface files between a C/C++ library and the Python.

Example

A simple C-function written as a Python extension that takes a single PyObject as input, passing it into a C-function my_function(variant) and outputting the result as a PyObject.

PyObject* Py_my_function(PyObject *self, PyObject *args)
{
PyObject* py_input;
if(!PyArg_ParseTuple(args, "O", &py_input))
{
return NULL;
}
variant input = core_python_py_to_variant(py_input);
variant rtn = my_function(input);
PyObject* py_rtn = core_python_variant_to_py(rtn);
variant_free(input);
variant_free(rtn);
return py_rtn;
}

Of course, if you use the C++ interface you will not need to free the variant objects after usage.
