arnovich_core Documentation

This is the Arnovich Core lib. This library contains some of the common functionality used by other projects.

The library is broken in to a few sub-module:

  • core_debug: simple debugging functionality
  • core_python: a series of small helper functions for interfacing with Python.
  • core_srv: a very simple push, pop, subscribe server with a C and Python interface.
  • core_types: common variant and matrix types with C and C++ bindings and easy Python interface.
  • build: common build scripts

The library is release under the BSD-3 license.
